Complete branding suite (banner, business card, and readme) for personal use over internet (e.g. GitHub profile readme).
 
## Usage

// Replace ```USER_NAME``` by your GitHub username.

// If you want the exact same template, <a href="mailto:info@jishan.codes">contact! </a>
- Create repository https://github.com/USER_NAME/USER_NAME (Profile readme repository)
- Add your files (Only PNGs are required) in proper directory order (/complete-branding-{banner, readme, card}/complete-branding-{banner, readme, card1, card2})
- In readme file, add the lines:
  - For banner:

    ```
    <img src="https://github.com/USER_NAME/USER_NAME/blob/main/complete-branding-banner/complete-branding-banner.png" width=900, height=1200 />
    ```
  - For card:
  
    ```
    <img src="https://github.com/USER_NAME/USER_NAME/blob/main/complete-branding-card/complete-branding-card1.png" width=600, height=350 />
    <img src="https://github.com/USER_NAME/USER_NAME/blob/main/complete-branding-card/complete-branding-card2.png" width=600, height=350 />
    ```
  - For readme (optional):
    ```
    <img src="https://github.com/USER_NAME/USER_NAME/blob/main/complete-branding-readme/complete-branding-readme.png" width=1200, height=900 />
    ```
